- Evidence: `D:/420-system/.tmp_acsfx_purge/diff_root_cause.json`.

- 6 leads had fabricated emails matching pattern `<lower(first)>.<lower(last)>@<domain>` (Match-Trade, Brokeree, Praxis, The5ers, iFX EXPO, Finance Magnates).

- Root cause: an earlier iteration of `insert_real_leads.py` synthesized emails from company names. The on-disk script was edited later to use `None`, but the synthesized values had already been committed.
- Cleanup: `D:/420-system/.tmp_acsfx_purge/cleanup_fabricated.py` ran a whitelist-based `UPDATE` against both `sales_leads` and `contacts`. Whitelist kept: FTMO (`support@ftmo.com` / `+420910920310`), Contentworks Agency (`hello@contentworks.agency`), CISI (`customersupport@cisi.org` / `+442076450777`), Sumsub (`+442038827770`), Brokeree (`+35725011886` phone only).
