## 3. EXACT CONTRACTS (verified)

### Email send (2 calls; tenant_id = SLUG `zateceptionist`, header `Content-Type: application/json`, auth none)
```
POST /webhook/manual-email/create-campaign
{ tenant_id, name, subject, body_html, recipients:[{email,name?,company?,custom_variables?}], body_text?, from_name?, from_email? }
→ { success, campaign_id, total_recipients, status:'draft', smtp_configured }
POST /webhook/manual-email/start-campaign  { tenant_id, campaign_id }  → status:'sending'  (ME.PROC delivers ≤3min)
POST /webhook/manual-email/get-status      { tenant_id, campaign_id }  → counts (poll to confirm)
```
Merge tokens in subject/body_html: `{{first_name}} {{name}} {{company}}` + custom vars. Sends + logs to `outbound_messages {channel:'email',recipient_identifier,status,...}`.

### Guess-email (enrichment; sync, read-only, no tenant_id)
`POST /webhook/guess-email { domain, contact_name? }` → `{ found, email, method }`.

### VAPI interview (auto-dial)
**Step A — seed** `hr_ai_interviews` (NOT NULL: `tenant_id`(UUID), `application_id`, `candidate_id`(must have `.phone`), `job_requisition_id`(must have `.job_title`)) + `questions` jsonb `[{question, expected_keywords?}]`, `status:'scheduled'`.
**Step B —** `POST /webhook/hr/ai-interview/start-call { interview_id }`. Workflow joins phone/title/creds, places call, stamps `vapi_call_id` + `status:'in_progress'`.
**Receiver `0VEBSpO63nEiR1xh`** (auto, matched on `vapi_call_id`): writes `transcript, recording_url, status, call_ended_at, actual_duration_seconds`, then (LLM-scored) `ai_score, question_scores, strengths, concerns, recommendation, ai_reasoning, completed_at`; mirrors score onto `hr_job_applications`.
**Disclosure (verbatim, already present):** firstMessage = *"Hi {first}, this is the AI interviewer from {company}. I have a few questions about your background and the {job_title} position. Is this a good time?"* — satisfies AI-disclosure; harden here if stricter consent wanted.

### Logs to WRITE
- `hr_recruitment_outreach` (NOT NULL: `tenant_id`, `application_id`): set `candidate_id, channel('email'/'voice'), message_type, subject, body, status('sent'), sent_at, provider, ai_generated`.
- `hr_job_applications` outreach cols: `outreach_status, outreach_channel, outreach_message, outreach_sent_at` (+ `outreach_replied_at`, `follow_up_count`).
